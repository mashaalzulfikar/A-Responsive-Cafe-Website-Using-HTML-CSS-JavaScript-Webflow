# A Responsive Cafe Website Using HTML, CSS, JavaScript, & Webflow

## Project Overview
This project is aimed at developing a responsive website for "Cafe 21CS". The website was created to establish an engaging online presence for a modern cafe and bistro, enabling potential customers to easily access information about the cafe's offerings, story, and contact details.

## Features
* **Responsive Design**: The website is optimized for various screen sizes, ensuring seamless browsing on desktops, tablets, and mobile devices.
* **Interactive Navigation**: The site provides an interactive experience across "Home", "Menu", "About", and "Contact" pages.
* **Detailed Menu**: The menu is organized into categories—including Specials, Pizzas, Gourmet Pizzas, Classics, Children, Dessert, and Sides, with a direct "Message on Facebook" button for inquiries and orders.
* **Integrated Tools**: Features include a contact form for inquiries, a Google Maps embed for location details, and a dedicated FAQ section to address common customer queries.
* **Engaging Story**: The "About" section highlights the cafe's family-run identity, the inspiration behind the business, and the head chef's background.

## Technologies Used
* **HTML**: Used to structure the content and layout.
* **CSS**: Used to style the appearance, fonts, colors, and layout.
* **JavaScript**: Implemented to add interactivity and dynamic features, such as animated links and responsive navigation.
* **Webflow**: Used to build responsive layouts visually and streamline the design process.
